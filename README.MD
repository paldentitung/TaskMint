# TaskMint - Full Stack Productivity App

## Stack

- **Frontend:** React + Tailwind CSS
- **Backend:** Express.js
- **Storage:** File system (JSON)
- **Extras:** Framer Motion for animations, React Icons, LocalStorage (optional for user data)
